AI weekend assignment: predicting chaotic time series

The Mackey-Glass equation (MGE) is the following Delay Differential Equation (DDE):

\[ \frac{dx_{t}}{dt} = \beta \frac{x_{t-\tau}}{1 + x_{t-\tau}^n} - \gamma x_t \]

At certain parameter values and initial conditions, the solution of the MGE is a chaotic time series. Particularly, we are interested in predicting the solution of the MGE when

\[ \begin{align*} \gamma &= 1.0 \\ \beta &= 2.0 \\ \tau &= 2.0 \\ n &= 9.65 \\ \end{align*} \]

At these parameter values and an initial condition and constant history of 1.1, the solution on $t\in\left[0, 180\right]$ looks like this:

using DifferentialEquations
using Plots

const β = 2.0
const τ = 2.0
const γ = 1.0
const n = 9.65

function mackey_glass(u, h, p, t)
  β * h(p, t - τ) / (1 + h(p, t - τ)^n) - γ * u
end

lags = τ
u0 = 1.1
h(p, t) = u0

t_span = (0.0, 180.0)
problem = DDEProblem{false}(mackey_glass, u0, h, t_span; constant_lags=lags)
alg = MethodOfSteps(Tsit5())

sol = solve(problem, alg)

plot(sol)
